### 代币经济学 - 核心数学

```
代币价值捕获 = f(协议收入, 代币销毁, 质押收益)

需要建模:
• 通胀率动态方程
• 质押收益率曲线
• 多池耦合效应
• 治理投票动力学

工具: 常微分方程、动力系统理论
```

### AMM数学

```
不变函数设计: φ(x₁, x₂, ..., xₙ) = k

常见类型:
• 加法不变: x + y = k           (恒定和)
• 乘法不变: x · y = k           (恒定乘积) ← Uniswap V2
• 混合不变: x·y + x·z + y·z = k ( stableswap) ← Curve
• 指数不变: xᵃ · yᵇ = k          (广义)

核心问题:
• 无常损失计算
• 流动性聚集算法
• 滑点优化
• 多资产扩展

工具: 凸优化、博弈论
```

### 借贷协议

```
利率模型: r = base + utilization × slope

稳定利率: 低利用率时
 slope = 0
斜率利率: 高利用率时  
 slope > 0

风险建模:
• 抵押率动态调整
• 清算引擎设计
• 坏账准备金

工具: 随机过程、信用风险模型
```
