## 工具包说明

### 核心功能

本工具包用于分析金融时间序列的非有效市场特征，包括：

| 功能 | 说明 |
|:---|:---|
| 波动率聚类 | 检测高波动期是否会持续 |
| 长记忆性 | 判断价格是否有趋势或均值回归特征 |
| 尾部风险 | 识别极端收益事件的频率 |
| 多尺度分析 | 从分钟到月线全面分析 |
| 市场状态诊断 | 自动判断当前市场类型 |
| 策略建议 | 根据分析结果给出操作建议 |

### 理论基础

本工具基于以下理论：
- **ARCH/GARCH模型**: 波动率聚类效应
- **R/S分析**: Hurst指数计算
- **DFA**: 去趋势波动分析
- **分形市场假说**: 多尺度记忆性

## 快速开始

### 1. 导入数据

```matlab
% 方式1: 直接加载
load('..\matlab_data\BTC_1h.mat')

% 方式2: 从工作区获取变量
% 确保工作区中有: timestamp, open, high, low, close, volume
% 或 raw 结构体
```

### 2. 一键分析

```matlab
cd E:\NEMT Simulator\matlab\scripts
load('..\matlab_data\BTC_1h.mat')
NEMT_FullAnalysis        % 运行完整分析流程
```

### 3. 分步分析

```matlab
% 图表分析
NEMT_QuickPlot           % K线图 + 成交量
NEMT_MainAnalysis        % 收益率分析
NEMT_Dashboard          % 综合仪表盘
NEMT_VolatilityClustering  % 波动率聚类
NEMT_MultiScaleAnalysis  % 多尺度相关
NEMT_HurstAnalysis       % Hurst分形

% 生成报告
report = NEMT_IntelligenceReport  % 智能诊断
NEMT_ExportReport(report)         % 导出报告
```

## 报告解读

### 核心指标速查

| 指标 | 说明 | 参考值 |
|:---|:---|:---|
| Hurst指数 | 长期记忆性 | H>0.5趋势，H<0.5反转 |
| ARCH效应 | 波动率聚类 | >0.3为强聚类 |
| 峰度 | 尾部厚度 | >5为厚尾 |
| 偏度 | 分布对称性 | <0为左偏(跌) |
| VaR | 在险价值 | 95%置信度最大损失 |

### 市场状态速判

| H | ARCH | 峰度 | 市场类型 | 策略 |
|:---:|:---:|:---:|:---|:---|
| 高 | 高 | 高 | 混沌市场 | 对冲+轻仓 |
| 高 | 低 | 正常 | 稳定趋势 | 趋势跟踪 |
| 低 | 高 | 高 | 波动市场 | 反转+买期权 |
| 低 | 低 | 正常 | 震荡市场 | 区间操作 |
| ≈0.5 | - | - | 随机市场 | 观望 |

### 风险等级速查

| 风险等级 | 颜色 | 说明 |
|:---|:---:|:---|
| 低风险 | 🟢 | 可正常操作 |
| 中低风险 | 🟡 | 适当关注 |
| 中高风险 | 🟠 | 控制仓位 |
| 高风险 | 🔴 | 减仓观望 |

## 注意事项

1. **数据要求**: 建议数据长度 > 500 点以保证分析准确性
2. **时间格式**: 支持 Unix时间戳、Excel日期、MATLABdatenum
3. **局限性**: 历史分析不代表未来表现
4. **风险提示**: 请结合基本面和市场情绪综合判断
