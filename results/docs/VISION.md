# NEMT 系统愿景 (VISION)

> 最后更新: 2026-04-19

## 一、北极星指标

| 指标 | 目标值 | 当前值 | 差距 |
|------|--------|--------|------|
| **夏普比率 (Sharpe)** | > 1.5 | - | - |
| **最大回撤 (Max Drawdown)** | < 20% | - | - |
| **胜率 (Win Rate)** | > 55% | - | - |
| **盈亏比 (Profit Factor)** | > 1.5 | - | - |

## 二、核心原则

```
1. 风控优先级高于收益
   → 任何时候，回撤保护优先于收益最大化

2. 结果导向
   → 每一个改动都要能用数字证明其有效性

3. 策略生态健康
   → 策略不是越多越好，优胜劣汰才是核心

4. 透明可追溯
   → 所有决策都要记录原因，所有结果都要可复现
```

## 三、边界约束

- **不做**: 真实交易（目前仅回测验证）
- **不做**: 单一策略依赖（必须多策略组合）
- **不做**: 黑盒优化（所有参数调整必须有理论依据）

## 四、成功定义

> **系统整体夏普 > 1.5 且 最大回撤 < 20%，持续稳定 3 个月**
