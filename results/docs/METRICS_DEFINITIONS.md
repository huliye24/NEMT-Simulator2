# 指标定义 (METRICS_DEFINITIONS)

> 最后更新: 2026-04-19

## 一、核心指标

### 1. 夏普比率 (Sharpe Ratio)

```python
夏普比率 = (年化收益率 - 无风险利率) / 年化波动率

默认值:
- 无风险利率 = 2% (年化)
- 交易日 = 252天/年
```

**计算频率**: 每日收盘后
**数据来源**: 回测引擎输出的每日收益率序列

### 2. 最大回撤 (Max Drawdown)

```python
最大回撤 = max(峰值 - 谷值) / 峰值 × 100%

阶段:
- 绿色: < 5%
- 黄色: 5-10%
- 橙色: 10-15%
- 红色: > 15% (触发告警)
```

**计算频率**: 每日收盘后
**告警阈值**: 15%

### 3. 胜率 (Win Rate)

```python
胜率 = 盈利交易数 / 总交易数 × 100%
```

**计算频率**: 每次交易完成后
**健康范围**: > 45%

### 4. 盈亏比 (Profit Factor)

```python
盈亏比 = 总盈利 / 总亏损

示例:
- 盈亏比 2.0 = 每亏损1元，盈利2元
```

**计算频率**: 每月
**健康范围**: > 1.5

### 5. 策略存活率 (Strategy Survival Rate)

```python
存活率 = (期末活跃策略数 / 期初活跃策略数) × 100%

计算周期: 90天
```

## 二、策略指标

### 策略评分 (Total Score)

```python
总分 = 盈利能力×0.3 + 一致性×0.2 + 风险调整×0.3 + 适应性×0.2

评分范围: 0-100
淘汰线: < 30
```

## 三、告警阈值

| 指标 | 告警阈值 | 动作 |
|------|----------|------|
| 最大回撤 | > 15% | 降低仓位50% |
| 单日亏损 | > 5% | 进入防御模式 |
| 夏普比率 | < 0 | 暂停策略 |
| 连续亏损 | > 5天 | 触发评估 |

## 四、数据来源

| 指标 | 来源 | 频率 |
|------|------|------|
| 收益率 | 回测引擎 | 每日 |
| 交易记录 | 执行层 | 实时 |
| 策略评分 | 策略服务 | 每日 |
| 系统状态 | 大脑层 | 实时 |
