## 三、告警级别定义

| 级别 | 颜色 | 条件 |
|------|------|------|
| 严重 (Critical) | 🔴 红色 | 回撤 > 15% 或 夏普 < -1 |
| 警告 (Warning) | 🟡 黄色 | 回撤 > 10% 或 夏普 < 0 |
| 注意 (Notice) | 🟢 绿色 | 其他正常波动 |

## 四、自动告警规则

```python
# 严重告警
if drawdown > 15:
    send_alert("CRITICAL", f"回撤超限: {drawdown}%")

if sharpe < -1:
    send_alert("CRITICAL", f"夏普异常: {sharpe}")

# 警告告警
if drawdown > 10:
    send_alert("WARNING", f"回撤偏高: {drawdown}%")

if sharpe < 0:
    send_alert("WARNING", f"夏普为负: {sharpe}")
```
