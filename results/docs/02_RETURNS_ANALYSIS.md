## 二、关键指标解读

### 收益率统计

```matlab
统计量:
  mean(returns)   - 平均日收益率
  std(returns)   - 收益率标准差(波动率)
  skewness(returns) - 偏度
  kurtosis(returns)  - 峰度
```

**各指标含义:**

| 指标 | 计算公式 | 正常范围 | 异常含义 |
|:---|:---|:---|:---|
| 均值 | Σr/n | -0.1%~+0.3% | 正→上涨市场，负→下跌市场 |
| 标准差 | √(Σ(r-mean)²/n) | 1%~5% | 高→波动剧烈 |
| 偏度 | E[(r-μ)³/σ³] | -0.5~+0.5 | 负→左偏(跌)，正→右偏(涨) |
| 峰度 | E[(r-μ)⁴/σ⁴] | 3~5 | >5→厚尾(极端事件多) |

## 三、市场状态判断

### 典型分布形态对比

#### 正常市场
```
特征:
• 偏度接近0 (对称)
• 峰度接近3 (正态)
• 极端收益很少

解读:
市场没有明显的方向偏好，
价格波动相对"温和"
```

#### 恐慌市场
```
特征:
• 偏度 < -0.5 (明显左偏)
• 峰度 > 7 (极厚尾)
• 左侧出现多个极端负收益

解读:
负收益极端事件频繁发生，
市场恐慌情绪严重
```

#### 狂热市场
```
特征:
• 偏度 > +0.5 (明显右偏)
• 峰度 > 5 (厚尾)
• 右侧出现多个极端正收益

解读:
正收益极端事件频繁发生，
市场过于乐观
```

## 四、实战应用

### 波动率交易策略

```
波动率阶段      特征              策略
──────────────────────────────────────���───────
低波动期   σ < 历史均值50%    买入期权(赌突破)
          │                  或观望
          ▼
波动率上升期  σ快速上升       追涨波动率
          │                  或顺势交易
          ▼
高波动期   σ > 历史均值80%    卖出期权(赌回归)
          │                  或对冲
          ▼
波动率下降期  σ快速下降       平仓了结
          │                  
          ▼
(循环...)
```

### 收益率分布应用

| 观察到的分布 | 可能的市场状态 | 建议操作 |
|:---|:---|:---|
| 尖峰厚尾 | 消息市/恐慌市 | 设置宽止损，使用期权对冲 |
| 右偏厚尾 | 上涨狂热 | 逢高减仓，警惕反转 |
| 左偏厚尾 | 下跌恐慌 | 空头为主，避免抄底 |
| 对称正态 | 正常市场 | 区间震荡操作 |

## 五、数据案例

### BTC正常波动
```
均值:     +0.15%  (日均小幅上涨)
标准差:   3.5%    (日波动3.5%)
偏度:     -0.12   (略左偏)
峰度:     4.2     (轻微厚尾)

解读: 波动正常，极端事件略多于正态分布
```

### BTC高波动期
```
均值:     +0.05%  (涨幅收窄)
标准差:   8.2%    (日波动8.2%)
偏度:     -0.85   (明显左偏)
峰度:     12.5    (极度厚尾)

解读: 市场极度不稳定，负收益极端事件频发
```

## 六、指标阈值参考

| 指标 | 安全值 | 警戒值 | 危险值 |
|:---|:---|:---|:---|
| 日标准差 | < 3% | 3%~5% | > 5% |
| 偏度 | -0.3~+0.3 | ±0.3~±0.7 | < -0.7 或 > +0.7 |
| 峰度 | < 5 | 5~7 | > 7 |
| 年化波动率 | < 30% | 30%~60% | > 60% |
