# 资源分配 (RESOURCE_ALLOCATION)

> 最后更新: 2026-04-19

## 一、计算资源分配

| 策略 | 计算时间占比 | 回测优先级 | 备注 |
|------|-------------|-----------|------|
| - | 暂无策略 | - | - |

## 二、资源计算公式

```
策略资源权重 = (策略评分 / 总评分) × 可用计算时间
```

## 三、分配规则

| 策略状态 | 计算资源 | 回测频率 |
|----------|----------|----------|
| ALIVE | 60% | 每日 |
| TESTING | 30% | 每周 |
| DORMANT | 10% | 每月 |
| DEAD | 0% | 无 |
