# Switchboard 学习计划

## 目标
在 30 天内掌握 Switchboard 预言机网络的核心能力，并找到盈利切入点。

---

## 第一周: 认知建立

### Day 1-2: 基础概念
```
任务:
- 阅读 https://docs.switchboard.xyz/how-it-works
- 理解 Oracle/Feed/Crank/Queue 概念
- 浏览 explorer.switchboardlabs.xyz

产出:
- 整理概念术语表
```

### Day 3-4: 生态探索
```
任务:
- 查看现有的数据 feeds
- 了解 SWTCH 代币经济
- 阅读 NCN 机制文档

产出:
- 现有 Feed 分类总结
- 代币经济分析
```

### Day 5-7: 质押体验
```
任务:
- 在 DEX 购买 SWTCH (小额)
- 在 Switchboard 尝试质押
- 观察奖励发放周期

产出:
- 质押操作记录
- 收益数据追踪
```
