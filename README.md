# NEMT Simulator V2
## 非平衡市场理论模拟器 (BTC实验版)

基于Notion理论中枢开发的完整NEMT交易系统实现。

### 核心数学模型

改进的非线性薛定谔方程 (NLS):

```
i∂ψ/∂t + α∂²ψ/∂x² + β|ψ|²ψ = η(x,t)
```

**参数说明:**
- ψ (psi): 市场状态复振幅
- α (alpha): 扩散系数（市场流动性）
- β (beta): 非线性系数（情绪/杠杆/羊群效应）
- η (eta): 噪声（外部扰动/随机交易）

### 核心模块说明

#### 1. nemt_core.py - 核心模拟器
NLS方程的数值求解，包含：
- 状态初始化与复振幅转换
- 时间演化（Euler方法）
- 频谱分析（FFT）
- 谱宽计算
- 共振检测

#### 2. nemt_signals.py - 信号指标模块
实现第二章、第六章定义的量化指标：
- **DCI (方向一致性指数)**: 测量市场噪声强度
  - DCI > 0.70: 低噪声，市场方向明确
  - DCI < 0.55: 高噪声，市场混乱
- **涡旋检测**: 四个条件的量化
  - BBW < 20%分位数（波动率收窄）
  - 成交量均匀分布（上涨/下跌各40-60%）
  - OI高位走平（变化<5%且水平>80%分位）
  - 资金费率零轴摆动（7日均值<0.01%）
- **随机共振检测**: 三条件量化
  - 长周期临界点（MVRV、NUPL、LTH等）
  - 短周期噪声适中（DCI波动率0.08-0.15）
  - 潜在触发因子（宏观/链上事件）

#### 3. nemt_onchain.py - 链上数据模块
实现第五章定义的链上指标：
- **MVRV / MVRV Z-score**: 市值/已实现市值比率及Z值
- **NUPL**: 未实现净损益指标
  - < 0: 投降区
  - 0-0.25: 希望/恐惧
  - 0.25-0.50: 乐观/焦虑
  - 0.50-0.75: 信念/否认
  - > 0.75: 欣快/贪婪
- **交易所余额**: 余额变化趋势
- **LTH/STH供应占比**: 长期/短期持有者结构
- **稳定币指标**: 稳定币市值变化
- **链上健康度评分**: 综合0-10评分系统

#### 4. nemt_state_machine.py - 四相位状态机
实现市场四相位自动识别：
- **相位A (高噪声混乱期)**
  - 策略: 仅持有底仓，不做短线
  - 最大仓位: 20%
  - 关注: DCI回升、涡旋形成
- **相位B (涡旋蓄力期)**
  - 策略: 识别边界，预设条件单，不预判方向
  - 最大仓位: 50%，杠杆≤1x
  - 关注: 涡旋成熟度、突破确认
- **相位C (临界爆发前夜)**
  - 策略: 提高敏感度，敢于追入
  - 最大仓位: 70%，杠杆≤2x
  - 关注: 触发事件兑现、量能持续性
- **相位D (趋势运行期)**
  - 策略: 持仓为主，回调加仓
  - 最大仓位: 100%，杠杆≤1x
  - 关注: DCI高位回落、SNR萎缩

#### 5. nemt_execution.py - 执行框架
实现第七章定义的完整交易流程：

**第一步: 预测**
- 建立方向假设（看涨/看跌/中性）
- 确定周期级别（日内/波段/趋势）
- 开启狩猎模式条件判断

**第二步: 信号**
- 涡旋突破信号
- 随机共振触发信号
- 趋势回调信号
- 宏观链上共振信号

**第三步: 验证**
- 成交量确认（>1.5倍均值）
- OI增加确认
- 时间确认（突破保持）
- 资金费率确认
- 链上确认（交易所净流出）

**第四步: 加仓**
- 回踩确认加仓（回调2-8%）
- 突破确认加仓（突破2%）
- 概率提升加仓（P_trend增加≥10%）

#### 6. nemt_risk.py - 风险管理模块
实现第八章定义的风控系统：
- **仓位管理**: 相位驱动的动态仓位
- **止损管理**: ATR适应型止损 + 移动止损
- **杠杆管理**: 安全杠杆计算
- **资金曲线管理**: 回撤控制（绿/黄/橙/红四级）
- **心理风控**: 冷静期规则
- **波动率警戒**: ATR异常放大警告

### 三个核心实验

1. **噪声扫描实验**: 研究不同噪声水平下的谱结构响应
2. **非线性扫描实验**: 研究情绪/杠杆效应对市场的影响
3. **真实vs模拟对比**: 比较原始价格与模拟后的差异
