## Summary
- Add complete NEMT Quant OS MVP framework with 9 core modules
- Event-driven backtesting engine
- Strategy evolution system
- 4-level risk management

## Modules
- MarketLayer: data loading, regime detection
- SignalLayer: RSI, ATR, MACD, Bollinger Bands
- StrategyLayer: Trend, MeanReversion, Momentum strategies
- ExecutionLayer: order management, position tracking
- RiskLayer: 4-level risk mode system (NORMAL/CAUTION/DEFENSE/SHUTDOWN)
- BrainLayer: strategy weight allocation
- EvolutionLayer: strategy scoring and eviction
- Dashboard: performance visualization
